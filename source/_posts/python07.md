 ## 函数
 函数：是组织好的，可重复使用的，用来实现特定功能的代码段。
 ```python
name = "xiaobaixuePython"
length = len(name)
print(length)
```
```
16 
```
为什么随时都可以使用len()统计长度？
因为：len（）是Python内置的函数：
- 是提前写好的
- 可以重复使用
- 实现统计长度这一特定功能的代码段
